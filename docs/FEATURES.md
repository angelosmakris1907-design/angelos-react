# Angelos Features

## Core
- Add, edit, delete, complete tasks
- Undo delete
- Save tasks locally
- Import/export backup

## Voice
- Add tasks by voice
- Read tasks aloud
- Ask what to do next
- Ask what is due today/tomorrow/overdue

## Reminders
- Due dates and times
- Relative reminders: “in 10 minutes”
- Automatic voice reminders
- Browser notifications

## Organisation
- Priorities
- Categories
- Custom categories
- Weekly agenda
- Today dashboard

## Notes
- Create notes
- Read notes
- Search notes
- Edit/delete notes
