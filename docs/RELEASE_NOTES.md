# Angelos MVP v1.0 Release Notes

## Version
1.0 MVP

## What Angelos Does
Angelos is a voice-first planner that helps users manage tasks, reminders, notes, categories, and daily planning.

## Main Features
- Voice task creation
- Natural language reminders
- Recurring tasks
- Notes
- Custom categories
- Daily briefing
- Today dashboard
- Weekly agenda
- Browser notifications
- Import/export backup
- Settings

## Known Limits
- Reminders work best while the app is open
- Phone call reminders need a future backend
- Speech recognition depends on browser support
