# Angelos MVP Demo Script

## 1. Introduction
Angelos is a voice-first planner designed to help people organise tasks, reminders, notes, and daily planning with as little typing as possible.

## 2. Voice Task
Say:
"Study chemistry tomorrow at 6"

Show:
- Task appears
- Due date appears
- Time appears

## 3. Reminder
Say:
"Remind me in 2 minutes to call George"

Show:
- Upcoming Soon
- Voice reminder
- Browser notification

## 4. Categories
Say:
"Create category university"
"Move chemistry to university"
"Read my university tasks"

## 5. Notes
Say:
"Remember George prefers morning appointments"
"Search notes for George"

## 6. Briefing
Click:
Morning Briefing

Show:
- Briefing text
- Today Dashboard

## 7. Agenda
Say:
"Read my agenda"

## 8. Closing
This is Angelos MVP v1.0: a working voice-first assistant prototype.
